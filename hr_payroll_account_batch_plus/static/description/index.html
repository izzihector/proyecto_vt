HR Payroll Batch Journal Entry

Allow to create one journal entry for payslip batch

Odoo HR Payroll Batch Journal Entry for create one journal entry for each payslip batch.

This Extension Allows

  • Select accounting date for batch.
  • Generate one journal entry for all payslips.
  • Restrict confirm payslip if it linked to batch.
  • Create Batch for employees in same company only.

Payslip Batch View

Determine accounting date
Close payslip batch will add link to journal entry and set payslips as done

Payslip View

Restrict confirm payslip if it related to batch

Journal Entry

Close payslip batch to generate one journal entry contain lines for each payslip lines
